# pj_vendas_estag_exata
Esse é um repositório destinado a montagem do meu projeto de estágio da empresa exata tech, onde fui incluído a pouco tempo. O projeto consiste em montar uma pipeline de dados no Databricks, na arquitetura lakehouse, com a finalidade de estudar os conceitos de análise e arquitetura de dados e aplicá-los no projeto.
